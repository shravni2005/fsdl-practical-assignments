# Assignment 04

## Aim
To develop a simple Insurance Management Website using Node.js and Express with MongoDB for storing and managing insurance policy data.

## Technologies Used
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript

## Steps Performed
1. Set up the Node.js and Express project.
2. Created webpages for Home, About, Services, Insurance Plans, and Contact.
3. Connected the application with MongoDB to manage insurance data.
4. Implemented routing and server-side functionality using Express.
5. Tested the application and captured output screenshots.

## Output
The Insurance Website allows users to view insurance plans and interact with the website through a functional Node.js and Express backend. Output screenshots are available in the `output-screenshots` folder.
